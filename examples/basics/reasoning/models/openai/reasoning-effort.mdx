title: OpenAI gpt-5-mini with reasoning effort

<Steps>

  <Step title="Create a Python file">
    ```bash
    touch reasoning_effort.py
    ```
  </Step>


  <Step title="Add the following code to your Python file">
    ```python reasoning_effort.py
    from agno.agent import Agent
    from agno.models.openai import OpenAIChat
    from agno.tools.duckduckgo import DuckDuckGoTools

    agent = Agent(
        model=OpenAIChat(id="gpt-5-mini", reasoning_effort="high"),
        tools=[DuckDuckGoTools(search=True)],
        instructions="Use tables to display data.",
        markdown=True,
    )

    agent.print_response("Write a report comparing NVDA to TSLA", stream=True)
    ```
  </Step>

  <Snippet file="create-venv-step.mdx" />

    ```bash
    pip install -U agno openai ddgs
    ```

  </Step>

    </CodeGroup>
  </Step>
  <Step title="Run Agent">
    <CodeGroup>


    </CodeGroup>
  </Step>
