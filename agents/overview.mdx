description: "Agents are AI programs that reason, use tools, and maintain state across interactions."
mode: wide
---

Agents are a stateful control loop around a stateless reasoning core: a model that uses tools in a loop, guided by instructions. Add memory, knowledge, storage, human-in-the-loop, and guardrails as needed.
