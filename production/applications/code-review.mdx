## Overview

A code review agent that analyzes pull requests, provides context-aware suggestions, and enforces coding standards and best practices.

## Planned Features

- Pull request analysis and summary
- Bug detection and security vulnerability scanning
- Style and convention enforcement
- Performance improvement suggestions
- Test coverage recommendations
- Documentation gap identification

## Use Cases

- Automated PR reviews
- Junior developer mentorship
- Security-focused code audits
- Codebase consistency enforcement
- Pre-merge quality gates
