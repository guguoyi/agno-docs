## Overview

A workflow that analyzes sales call recordings to extract insights, identify objections, and score conversation quality for coaching purposes.

## Planned Features

- Call transcription and speaker diarization
- Key moment identification
- Objection detection and categorization
- Talk-to-listen ratio analysis
- Next step extraction
- Deal stage recommendation
- Coaching feedback generation

## Workflow Steps

```
1. Receive call recording
2. Transcribe with speaker labels
3. Identify key discussion topics
4. Extract objections and responses
5. Analyze conversation dynamics
6. Score call quality
7. Generate coaching insights
```

## Use Cases

- Sales team coaching
- Deal review meetings
- Win/loss analysis
- Onboarding new reps
- Methodology compliance
