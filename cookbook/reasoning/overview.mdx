Reasoning-enabled agents solve complex problems through step-by-step thinking. Use dedicated reasoning models or add reasoning capabilities to any model.

```python
from agno.agent import Agent
from agno.models.openai import OpenAIChat

agent = Agent(
    model=OpenAIChat(id="o3-mini"),
)

agent.print_response(
    "A bat and ball cost $1.10. The bat costs $1 more than the ball. How much does the ball cost?",
    stream=True,
    show_full_reasoning=True,
)
```

## Reasoning Approaches

| Approach | Best For | Models |
|----------|----------|--------|
| Native Reasoning | Complex problems | OpenAI o1/o3, DeepSeek-Reasoner |
| Extended Thinking | Visible reasoning | Claude with thinking, Gemini with thinking |
| Reasoning Agent | Any model | Add reasoning to GPT-4, Llama, etc. |

## Reasoning by Model

### OpenAI o3-mini

Native reasoning model.

```python cookbook/81_reasoning/models/openai/o3_mini.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

agent = Agent(model=OpenAIChat(id="o3-mini"))

agent.print_response(
    "Solve the trolley problem. Evaluate multiple ethical frameworks.",
    stream=True,
)
```

### Claude Extended Thinking

See Claude's reasoning process.

```python cookbook/81_reasoning/models/anthropic/basic_reasoning.py
from agno.agent import Agent
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(
        id="claude-sonnet-4-5",
        thinking={"type": "enabled", "budget_tokens": 10000},
    ),
)

agent.print_response(
    "How many r's are in strawberry?",
    stream=True,
    show_full_reasoning=True,
)
```

### Reasoning with Any Model

Add reasoning to models without native support.

```python cookbook/81_reasoning/agents/default_chain_of_thought.py
from agno.agent import Agent
from agno.models.groq import Groq

agent = Agent(
    model=Groq(id="llama-3.3-70b-versatile"),
    reasoning_model=Groq(id="deepseek-r1-distill-llama-70b"),
)

agent.print_response(
    "Calculate the compound interest on $1000 at 5% for 3 years",
    show_reasoning=True,
)
```

## Reasoning with Tools

Combine reasoning with tool use.

```python cookbook/81_reasoning/tools/reasoning_tools.py
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.tools.yfinance import YFinanceTools

agent = Agent(
    model=Claude(
        id="claude-sonnet-4-5",
        thinking={"type": "enabled", "budget_tokens": 8000},
    ),
    tools=[YFinanceTools(stock_price=True, analyst_recommendations=True)],
)

agent.print_response(
    "Analyze NVDA as an investment. Consider technical and fundamental factors.",
    stream=True,
    show_full_reasoning=True,
)
```

## Reasoning Teams

Multi-agent reasoning.

```python cookbook/81_reasoning/teams/reasoning_finance_team.py
from agno.team import Team
from agno.agent import Agent
from agno.models.openai import OpenAIChat

analyst = Agent(
    name="Analyst",
    model=OpenAIChat(id="o3-mini"),
    role="Analyze the problem",
)

critic = Agent(
    name="Critic",
    model=OpenAIChat(id="gpt-4o"),
    role="Find flaws in the analysis",
)

team = Team(agents=[analyst, critic], mode="coordinate")
team.print_response("Should we invest in renewable energy stocks?")
```
